```
/pidginenglish

```

## USAGE
Send a GET request with 'q' as a key and a body of text as it's value...
```
http://www.translatefun.com/pidginenglish?q=Alice chased the big fat cat. Kimo went to the mall.
```

Get back json translated into Pidgin English...
```
{"text":"da kine chased da big fat da kine. da kine went to da da kine."}
```
